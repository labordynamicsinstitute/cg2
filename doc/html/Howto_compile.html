HowTo Compile the CG2 Source Code


Download the source code here.

If you are using the Intel compilers, first make sure that your environment is properly set-up.  Add the following lines into .bashrc, .cshrc, .login, or .profile depending on your default shell.  The exact path may differ slightly depending on your installation and the version of the Intel compiliers you are using.  This information is correct for the default install of Version 8.2 on  IA32 Linux.  The Sun compilers do not require any modification to your path.
source /opt/intel_fc_80/bin/ifortvars.sh
source /opt/intel_cc_80/bin/iccvars.sh
source /opt/intel/mkl72/tools/environment/mklvars32.sh
  1. Create a directory to hold the source code (suggest cg2f90).  
  2. Within cg2f90 create two subdirectories, one named src and the other named bin.  Within the src directory create two additional directories named; cg2_4v3 and groups.
  3. Copy the cg2.tar.gz file to the cg2_4v3 directory and the groups.tar.gz file to the groups directory.
  4. Un-tar each of the files (on Windows systems, WinZip should work).
    (Alternatively, the four previous steps above are done automatically if using the full repository dump from  http://repository.vrdc.cornell.edu/viewvc/cg2/trunk.tar.gz?view=tar. Download the file (trunk.tar.gz), un-tar it, and rename the trunk directory to cg2f90)
  5. If make is available on your system then you can use the Makefile to compile.  Otherwise you can use the Makefile as a guide to compile the program manually.  Type make -f <Makefile_xxxx>  while in both the cg2_4v3 and groups directory.  The binaries should now be created..
  6. Move each binary to the bin directory.
  7. Add the location of the binaries to your path.
The following lines will add the binaries to your path if they are installed in your home directory.  If you are installing the binaries for multiple users, the binaries could be placed in a standard location such as /usr/local/bin.
#Include the CG Binaries in the path
export PATH=$PATH:~/cg2f90/bin

You should now be ready to run cg2_4v3 and the grouping program, groups.
